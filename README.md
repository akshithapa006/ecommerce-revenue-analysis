# E-Commerce Revenue Analysis

Tools Used :
* PostgreSQL (pgAdmin)
* SQL
* Microsoft Excel

Description :
This project analyzes e-commerce transaction data to understand revenue trends, customer behavior, and product performance.

Key Analysis :
* Customer Lifetime Value (CLV)
* Revenue by product
* Monthly revenue trends
* Top products per year
* Customer contribution to revenue

Files :
* `ecommerce_revenue_analysis.sql` – SQL queries and analysis
* `ecommerce_revenue_analysis.xlsx` – Excel dashboard
